## 四、核心模块详解

### 4.1 MD5 修改 (`md5_changer.py`)

| 方式 | 原理 | 速度 | 画质影响 | 适用场景 |
|------|------|:----:|:--------:|----------|
| **尾部追加** | 文件末尾追加随机字节 | ⚡极快 | ❌ 无 | 快速去重，不改变文件内容 |
| **元数据修改** | 修改 EXIF / 视频标签 | 🔥快 | ❌ 无 | 轻度修改，保留原始画质 |
| **重编码** | ffmpeg/Pillow 重新编码 | 🐢慢 | ✅ 可调 | 彻底改变文件结构 |

```
接口: POST /api/md5/change
参数: { filepath, methods: ["append","metadata","reencode"], quality }
```

### 4.2 去水印 (`watermark_remover.py`)

| 水印类型 | 处理方式 | 技术 |
|----------|----------|------|
| **静态固定** | 逐帧同区域修复 | OpenCV inpaint (Navier-Stokes) |
| **动态移动** | 模板匹配追踪 → 逐帧修复 | 模板匹配 + inpaint |
| **图片水印** | 单图区域修复 | OpenCV inpaint / 高斯模糊 |

```
流程: 上传文件 → 预览帧框选水印 → 选择类型/方法 → 执行 → 下载结果
接口: POST /api/watermark/preview  →  预览帧
      POST /api/watermark/remove   →  执行去水印
```

### 4.4 在线下载 (`downloader.py`)

| 平台 | 标识 | 需要 |
|------|------|------|
| 抖音 | douyin | 浏览器 Cookie |
| 快手 | kuaishou | - |
| B站 | bilibili | - |
| YouTube | youtube | - |
| 小红书 | xiaohongshu | Cookie |
| 微博 | weibo | - |

```
接口: POST /api/download       →  单个下载
      POST /api/download/batch →  批量下载
      GET  /api/platforms      →  支持的平台列表
```

## 六、启动方式

### Web 界面
```bash
# 启动服务器
cd "D:/fanben/媒体处理体系/backend" && python server.py
# 浏览器打开
http://localhost:5000
```

### Hermes 对话
```
"启动媒体处理"        →  后台启动服务器
"处理这个视频的MD5"   →  调用 MD5 模块
"给视频去水印"        →  调出去水印面板
"给视频生成文案"      →  分析→生成文案
"下载这个抖音链接"    →  yt-dlp 下载
```
